# Local Setup

## Terminal
1. Install Junie CLI from https://junie.jetbrains.com
2. Setup command line:
   1. Authenticate to JetBrains using your jetbrains account.
   2. Generate an auth token at https://junie.jetbrains.com/cli
3. Add to environment variable `JUNIE_API_KEY`
           
## Ide
1. Install the Junie plugin
2. Authenticate via your JetBrains account.
